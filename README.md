# Искра

Искра — это программа для изучения слов. Ты делаешь карточки: на одной
стороне слово, на другой стороне перевод. Программа показывает карточки, и
ты повторяешь их каждый день.

Всё работает в браузере. Нет сервера, нет аккаунта. Твои карточки живут
только на твоём телефоне или компьютере. Можно работать без интернета.
Когда ты открываешь программу в первый раз, там уже есть маленькая колода
русских слов.

## Что умеет программа

- **Карточки.** Можно создавать, менять и удалять. У карточки есть слово,
  перевод, пример и заметка.
- **Колоды.** Карточки живут в колодах. Одна колода — один язык.
- **Повтор (SM-2).** Программа знает, когда показать карточку снова. Если
  слово трудное, она покажет его скоро. Если слово лёгкое — позже.
- **Урок.** Пробел переворачивает карточку. Кнопки 1, 2, 3, 4 — это оценка:
  Снова, Трудно, Хорошо, Легко.
- **Импорт и экспорт.** Можно загрузить слова из файла CSV или JSON и
  сохранить их обратно.
- **Звук.** Программа читает слово вслух (Web Speech API).
- **Статистика.** Сколько карточек надо повторить, сколько дней подряд ты
  учишься, карта активности.
- **Поиск.** Можно найти слово в колоде.
- **Тёмная и светлая тема.** Программа работает и на телефоне (PWA).

## Как запустить

```sh
npm install
npm run dev
```

Открой адрес, который покажет Vite (обычно http://localhost:5173).

## Как собрать

```sh
npm run build
```

Готовые файлы будут в папке `dist`. Пути в сборке относительные, поэтому
`dist` можно открыть на любом хостинге: GitHub Pages, Netlify, Vercel или
простой файловый сервер.

## Технологии

## In English

Iskra ("spark") is a spaced-repetition flashcard app for learning
languages. It runs fully in the browser — no backend, no sign-up, nothing
leaves your device. It works offline and installs as a PWA. Cards, decks,
an SM-2 review schedule, CSV/JSON import and export, text-to-speech,
statistics, search, and light/dark themes.
