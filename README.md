# Искра

Интервальные карточки для изучения языков. Всё работает в браузере — без сервера, без аккаунта, без облака. Каждая колода и каждый повтор живут в IndexedDB прямо на твоём устройстве, а приложение ставится как офлайн-PWA на телефон или компьютер.

Iskra ("spark") is a spaced-repetition flashcard app for language learning. It runs entirely in the browser: no backend, no sign-up, no data leaving your device. It installs as an offline-capable PWA and comes with a starter Russian deck on first launch.

## Что внутри / Features

- **Настоящий алгоритм SM-2** — ease factor, рост интервала, планирование по датам. Не заглушка «покажу завтра».
- **Колоды и карточки** — создание, переименование, удаление; лицевая/обратная сторона, пример, заметка.
- **Импорт и экспорт** — CSV и JSON, в обе стороны.
- **Сессия повтора** — переворот по пробелу, оценка клавишами 1–4 (Снова / Трудно / Хорошо / Легко), подсказки следующего интервала на кнопках.
- **Озвучивание** — Web Speech API, язык берётся из настроек колоды.
- **Статистика** — карточки к повтору, серия дней, удержание, тепловая карта активности за 26 недель.
- **Офлайн-PWA** — устанавливается, работает без интернета после первой загрузки.
- **Светлая и тёмная тема**, адаптивная вёрстка.

## Запуск / Running locally

```sh
npm install
npm run dev
```

Открой адрес, который покажет Vite (обычно http://localhost:5173).

## Сборка и деплой / Build & deploy

```sh
npm run build
```

Готовый статический сайт окажется в `dist/`. Пути в сборке относительные (`base: './'`) и маршрутизация на хэше, поэтому `dist/` разворачивается на любом статическом хостинге — GitHub Pages, Netlify, Vercel, обычный файловый сервер — без дополнительной настройки переписывания URL.

## Стек / Stack

React · TypeScript · Vite · Tailwind CSS · Dexie (IndexedDB) · vite-plugin-pwa · Vitest
